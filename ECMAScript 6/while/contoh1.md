# while loop

#### contoh 1
```javascript
// WHILE LOOP

let flag = true;
let i=1;

while (flag === true) {
   
    if (i % 2 === 1) {
        flag = false;
    }
    i++;
}
console.log(1 % 2); // 1
console.log(i);
```
